Una gran variedad de lenguajes de programación permiten crear objetos e interactuar con ellos. "Programación Orientada a Objectos" es el nombre de este paradigma. La manera de crear un nuevo objeto varía en función de la sintaxis de cada programa, sin embargo, la estructura suele ser similar:

Nombre_del_objeto = new Molde_del_objeto

Luego de esta asignación por medio del operador =, la variable Nombre_del_objeto permite al programador interactuar con el objeto creado. Para esto necesita hacer uso de otra herramienta.

Los métodos, son la manera habital de interactuar con objetos virtuales en un lenguaje de programación. Usualmente, para cada acción o atributo de un objeto, pueden existir dos tipos de métodos.

  • Métodos que obtienen (get) el valor actual de un atributo específico, por ejemplo, color o posición.
    • getColor(){ retornar.color = Nombre_del_objeto.color; }
  • Métodos que modifican (set) el valor actual de un atributo específico, por ejemplo, color o posición.
    • setColor(nuevoColor){ Nombre_del_objeto.color = nuevoColor; }
Si bien estos conceptos e interacciones se limitan a los lenguajes de programación que permiten orientación a objetos, es divertido imaginar cómo sería interactuar con los objetos del mundo real de la misma manera. Puede leer más información en este sitio.
